# rattish-core
Basic rattish interpreter in C to use as core.

### recent changes
- renaming from "rattish-c" to "rattish-core"
- migration main idea from old version of ratatuille
- cmake added as main builder for future developement
- npm package as commanding center as lazy and usefull shortcut
- new dev scripts for build, install, test and deployment


### status of progress
Project will be base for implementations in other languages and on various platforms.
Before next huge updates it's important to collect all old attempts. It can cause delay
but this can give some clarifications of final cut of __Rattish-core__. There is still 
much work to do.
